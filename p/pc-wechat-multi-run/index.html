PC 微信多开工具

微信多开 so easy~

2019-11-29
阅读时长: 1 分钟

PC 微信本身是不允许多开的，但是却有多开的方法。

方法是：第一步，退出当前的微信；第二步，快速连续多次启动微信。

这个人工操作还是有点难度，但是用脚本就简单多了。脚本代码如下：

@echo off
echo 请先退出微信 PC 客户端，再执行该程序！
pause
start "" "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"
start "" "C:\Program Files (x86)\Tencent\WeChat\WeChat.exe"

  • 保存脚本步骤：新建一个 .txt，把以上代码保存到 .txt 里边，然后把 .txt 修改为 .bat，双击就可以启动了。

  • 代码执行流程：1，杀掉微信的进程；2，连续启动两次微信。

  • 如何三开乃至更多开？把 start 这行代码多复制几行。

  • 注意：因为会杀进程，所以在使用脚本之前，你要先保证微信里边没有正在编辑的内容。
