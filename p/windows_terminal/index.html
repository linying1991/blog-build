Windows Terminal，官方的新终端程序

Windows Terminal 是一款新式、快速、高效、强大且高效的终端应用程序

2022-03-13
阅读时长: 1 分钟

官方介绍：

Windows 终端程序是一款新式、快速、高效、强大且高效的终端应用程序，适用于命令行工具和命令提示符，PowerShell和 WSL 等 Shell 用户。主要功能包括多个选项卡、窗格、Unicode、和 UTF-8 字符支持，GPU 加速文本渲染引擎以及自定义主题、样式和配置。

介绍

Windows Terminal 是 Windows 10 以后官方开源发布的一款终端，Windows 11 系统已经默认安装。

特点

我主要是因为一、二点而很喜欢使用

  • 好看：界面很美观，用了这个再也不想用回系统自带的 cmd 了
  • 右键菜单（安装后需重启资源管理器或重开机）
    • 桌面右键
    • 文件夹右键
    • 文件夹背景右键
  • 多标签页
  • 丰富的快捷键
  • 适用于命令行工具和命令提示符，PowerShell和 WSL 等
